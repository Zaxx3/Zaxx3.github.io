What is HTML?

HTML or HyperText Markup Language is the standard markup language used for documents designed to be viewd in a web browser.


What is CSS?

CSS or Cascading Style Sheets is a style sheet language used for altering the visual part of a document written in a markup language, usch as HTML.


What is JavaScript?

JavaScript is a programming language often used in web sites for animations, calculations and altering the technical part of sites


How the browser is loading and processing a web page?

Every page load starts when a user engages an initial request, when he clicks a link, submits a form etc.
After the request has been sent to the application server, it then goes trough a processing application.
When the request has been processed, an HTML response is sent to the users browser which then starts to process the DOM.
As soons as the DOM finishes loading, the browser renders the page.


How events work in JavaScript?

Events in JavaScript ocur everytime the user or the browser manipulate a page. Events are used to execute JS responses, every HTML
element contains a set of events which can be used to trigger JS code.
